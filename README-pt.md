## :inbox_tray: Como rodar & editar

É importante que você tenha ![Node.js](https://nodejs.org) instalado na sua máquina.

```bash
# Install Banana na sua máquina
$ yarn add banana -D

# Inicie a aplicação
$ yarn dev

# Agora você pode acessar localhost:3000 no seu navegador
```
